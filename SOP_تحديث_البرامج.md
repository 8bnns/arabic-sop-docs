# إجراء التشغيل الموحد (SOP) — تحديث البرامج

**الرقم المرجعي:** SOP-002  
**التاريخ:** مايو 2026  
**المسؤول:** فريق تقنية المعلومات  

---

## الغرض
ضمان تحديث جميع البرامج والتطبيقات المثبّتة بانتظام للحفاظ على الأمان والاستقرار والأداء.

## النطاق
يشمل هذا الإجراء نظام التشغيل macOS وجميع التطبيقات المثبّتة سواء من App Store أو من مصادر خارجية.

## المتطلبات الأولية
- صلاحيات المسؤول (Admin)
- اتصال بالإنترنت مستقر
- مساحة حرة لا تقل عن **10 GB**
- إغلاق جميع التطبيقات غير الضرورية

---

## خطوات التنفيذ

### 1. إنشاء نسخة احتياطية
- افتح **Time Machine** وابدأ نسخة احتياطية يدوية
- انتظر اكتمال النسخ قبل المتابعة

### 2. تحديث نظام التشغيل macOS
```bash
softwareupdate --list
```
- راجع قائمة التحديثات المتاحة
- ثبّت التحديثات:
```bash
softwareupdate --install --all
```

### 3. تحديث تطبيقات App Store
- افتح **App Store** ← تبويب **التحديثات**
- اضغط **تحديث الكل**
- انتظر اكتمال جميع التحديثات

### 4. تحديث Homebrew والحزم
```bash
brew update
brew upgrade
brew cleanup
```
- راجع أي تحذيرات أو تعارضات وعالجها

### 5. تحديث pip وحزم Python
```bash
pip3 install --upgrade pip
pip3 list --outdated
pip3 install --upgrade $(pip3 list --outdated --format=columns | awk 'NR>2 {print $1}')
```

### 6. تحديث npm والحزم العالمية
```bash
npm update -g
```

### 7. التحقق من البرامج الخارجية
- افتح كل تطبيق مثبّت يدوياً
- تحقق من وجود تحديثات عبر قائمة **Help → Check for Updates**
- نزّل وثبّت التحديثات إن وُجدت

### 8. إعادة التشغيل والتحقق
- أعد تشغيل الجهاز
- تحقق من أن جميع التطبيقات تعمل بشكل صحيح

### 9. توثيق التحديثات
- سجّل إصدارات البرامج المحدّثة في `سجل_العمليات`
- دوّن أي مشكلات واجهتها وكيفية حلّها

## التكرار
**أسبوعياً** للتحديثات الأمنية، **شهرياً** للتحديثات الكاملة.

---

## تنبيهات مهمة
> ⚠️ لا تُحدّث إصدارات البرامج في بيئة الإنتاج دون اختبارها في بيئة تطوير أولاً.  
> ⚠️ احتفظ دائماً بنسخة احتياطية قبل أي تحديث كبير.
