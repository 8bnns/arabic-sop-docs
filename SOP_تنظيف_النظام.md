# إجراء التشغيل الموحد (SOP) — تنظيف النظام

**الرقم المرجعي:** SOP-003  
**التاريخ:** مايو 2026  
**المسؤول:** فريق تقنية المعلومات  

---

## الغرض
تحرير المساحة التخزينية وتحسين أداء الجهاز من خلال حذف الملفات المؤقتة والمخبّآت والبيانات غير الضرورية بشكل آمن ومنتظم.

## النطاق
يُطبَّق على جميع أجهزة macOS في بيئة العمل.

## المتطلبات الأولية
- صلاحيات المسؤول (Admin)
- مراجعة الملفات قبل الحذف للتأكد من عدم أهميتها
- نسخة احتياطية حديثة

---

## خطوات التنفيذ

### 1. فحص المساحة الحالية
```bash
df -h /
du -sh ~/Downloads ~/Desktop ~/Documents
```
- سجّل المساحة المستخدمة قبل التنظيف للمقارنة

### 2. تفريغ سلة المهملات
```bash
rm -rf ~/.Trash/*
```
- أو من الواجهة: انقر بزر الماوس الأيمن على سلة المهملات ← **إفراغ سلة المهملات**

### 3. حذف ذاكرة التخزين المؤقت للنظام
```bash
sudo rm -rf /Library/Caches/*
rm -rf ~/Library/Caches/*
```

### 4. حذف ملفات السجلات القديمة
```bash
sudo rm -rf /private/var/log/asl/*.asl
rm -rf ~/Library/Logs/*
```

### 5. تنظيف Homebrew
```bash
brew cleanup --prune=all
brew autoremove
```

### 6. حذف ملفات pip المؤقتة
```bash
pip3 cache purge
```

### 7. تنظيف npm
```bash
npm cache clean --force
```

### 8. حذف تطبيقات Docker غير المستخدمة
```bash
docker system prune -af --volumes
```
> ⚠️ تنبيه: هذا الأمر يحذف جميع الحاويات والصور المتوقفة. تأكد من عدم حاجتك لها.

### 9. مراجعة مجلد Downloads
- افتح مجلد `~/Downloads`
- احذف الملفات القديمة والمؤرشفات التي لم تعد بحاجة إليها

### 10. تنظيف سطح المكتب
- نقل الملفات المتراكمة على سطح المكتب إلى مجلداتها المناسبة
- حذف الملفات المؤقتة

### 11. إيقاف تشغيل التطبيقات غير المستخدمة
```bash
# عرض العمليات الأكثر استهلاكاً للذاكرة
top -l 1 -o mem | head -20
```
- أغلق التطبيقات التي تستهلك موارد دون استخدام فعلي

### 12. فحص المساحة بعد التنظيف
```bash
df -h /
```
- قارن النتيجة مع القراءة الأولى وسجّل المساحة المُحرَّرة

## التكرار
**أسبوعياً** للتنظيف الخفيف، **شهرياً** للتنظيف الشامل.

---

## تنبيهات مهمة
> ⚠️ لا تحذف أي ملف بدون مراجعته أولاً.  
> ⚠️ الملفات في `/System` محمية — لا تتدخل فيها.  
> ⚠️ احتفظ بنسخة احتياطية قبل حذف أي بيانات كبيرة.
