# سجل العمليات

هذا المجلد يحتوي على سجلات تنفيذ إجراءات SOPs. يُسجَّل كل تنفيذ في ملف منفصل باسم يتضمن التاريخ ورقم الإجراء.

## تسمية الملفات

```
YYYY-MM-DD_SOP-XXX_اسم-الإجراء.md
```

**أمثلة:**
- `2026-05-05_SOP-001_صيانة-الجهاز.md`
- `2026-05-05_SOP-003_تنظيف-النظام.md`

## كيفية الاستخدام

1. انسخ `قالب_سجل_عملية.md` واسمّه وفق الصيغة أعلاه
2. عبّئ جميع الحقول أثناء أو بعد تنفيذ الإجراء
3. احفظ الملف في هذا المجلد
